🤖 AI Text Summarizer (MERN + AI)

An AI-powered Text Summarizer built using the MERN Stack and a free AI model. Users can enter long text, and the application generates a concise summary using an LLM.
---

🚀 Features

- ✨ AI-powered text summarization
- 📝 User-friendly React interface
- ⚡ Express.js backend API
- 🤖 Free AI model integration
- 🔄 Real-time API communication
- 📱 Responsive design

---
🛠️ Tech Stack

Frontend
- React.js
- Axios
- CSS

Backend
- Node.js
- Express.js

AI
- OpenRouter API
- Free LLM

⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Text-Summarizer.git
```

### 2. Install dependencies

Backend

```bash
cd server
npm install
```

Frontend

```bash
cd client
npm install
```

---

🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000
OPENROUTER_API_KEY=YOUR_API_KEY
```
